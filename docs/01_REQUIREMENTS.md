# 요구사항 정의

> 본 문서는 박사논문용 웹 기반 통계분석 솔루션의 기능/비기능 요구사항을 정의한다.  
> SPSS 26 및 PROCESS Macro v5 기능을 Python으로 대체하는 것을 기준으로 작성되었다.

---

## 1. 기능 요구사항

### 1.1 데이터 관리

| ID | 요구사항 | 우선순위 |
|---|---|---|
| FR-001 | 사용자는 Excel(.xlsx, .xls) 및 CSV 파일을 업로드할 수 있어야 한다 | 필수 |
| FR-002 | 업로드된 파일은 서버의 지정 경로에 저장되어야 한다 | 필수 |
| FR-003 | 사용자는 업로드된 데이터의 미리보기(상위 N행)를 확인할 수 있어야 한다 | 필수 |
| FR-004 | 시스템은 데이터 클린징(결측치 처리, 이상치 탐지, 역코딩 등)을 수행할 수 있어야 한다 | 필수 |
| FR-005 | 사용자는 클린징 규칙을 설정하고 적용 결과를 확인할 수 있어야 한다 | 필수 |

### 1.2 변수 설정

| ID | 요구사항 | 우선순위 |
|---|---|---|
| FR-006 | 사용자는 변수 유형(독립/종속/매개/조절)을 설정할 수 있어야 한다 | 필수 |
| FR-007 | 사용자는 각 변수 유형에 하위변수(하위 변수)를 매핑할 수 있어야 한다 | 필수 |
| FR-008 | 사용자는 각 하위변수 유형에 요인을 그룹핑하도록 매핑할 수 있어야 한다 | 필수 |
| FR-009 | 사용자는 각 요인에 설문항목들을 매핑할 수 있어야 한다 | 필수 |
| FR-010 | 사용자는 변수 매핑 결과를 저장하고 수정할 수 있어야 한다 | 필수 |
| FR-011 | 사용자는 [매핑] -> [요인분석] -> [모형 설정] 순서대로 분석을 수행할 수 있어야 한다 | 필수 |
| FR-012 | 요인매핑이 되면 자동으로 변수단위로 자동요인분석을 할 수 있어야 한다. | 필수 |
| FR-014 | 변수단위로 하위요인이 있을수도 있고 없을수도 있다.| 필수 |
| FR-015 | 요인을 다른 변수에 드래그 앤 드롭으로 이동하면서 변수를 옮길 수 있어야 한다. | 필수 |
| FR-016 | 요인은 멀티선택이 가능하고 일괄 드래그도 가능하다. | 필수 |

### 1.3 요인분석 (3단계)

| ID | 요구사항 | 우선순위 |
|---|---|---|
| FR-009 | 사용자는 요인분석 기준치를 웹 화면에서 설정할 수 있어야 한다 | 필수 |
| FR-010 | 시스템은 설정된 기준치에 따라 탐색적 요인분석을 수행해야 한다 | 필수 |
| FR-011 | 요인분석 결과가 기준치를 만족하도록 요인을 자동 정리(미달 항목 제거 등)해야 한다 | 필수 |
| FR-012 | 사용자는 자동 결과를 확정하거나, 수동으로 항목을 선택/제거할 수 있어야 한다 | 필수 |
| FR-013 | 요인분석 재실행이 가능해야 한다 (기준치 변경 또는 항목 수동 조정 후) | 필수 |
| FR-017 | 요인분석 자동 정리 결과가 기준치 변경에 따라 자동으로 변수단위로 분석이 재구성되어야 한다 | 필수 |

#### 요인분석 설정 옵션

| 옵션 | 선택값 | 기본값 |
|---|---|---|
| 요인추출 방법 | 탐색적 성분 / 주성분(PCA) | 주성분 |
| 요인회전 방법 | Varimax | Varimax |
| 고유값(Eigenvalue) | 사용자 입력 | ≥ 1.0 |
| 요인적재값(Factor Loading) | 사용자 입력 | ≥ 0.5 |
| 공통성(Communality) | 사용자 입력 | ≥ 0.4 |
| 분산설명력(Explained Variance) | 사용자 입력 | ≥ 60% |
| KMO | 사용자 입력 | ≥ 0.5 |
| 고정된 요인수(숫자 설정 가능) | 사용자 입력 | 고정값(기본값 1), 고정된 요인개수 선택 |
| 계수 출력 형식은 크기순 정렬 | 사용자 입력 | 크기순 정렬 |
| 작은계수 표기할지 표기하지 않을지는 선택할수 있고 값도 설정 | 사용자 입력 | 표기하지 않음 |

## 3. SPSS 26 대체 대응표

| SPSS 26 기능 | Python 대체 라이브러리 | 비고 |
|---|---|---|
| 탐색적 요인분석 | `factor_analyzer` | FactorAnalyzer 클래스 |
| 주성분분석 (PCA) | `scikit-learn` (PCA) | sklearn.decomposition.PCA |
| Cronbach's α | `pingouin` | pg.cronbach_alpha() |
| Pearson/Spearman 상관 | `scipy.stats` / `pandas` | pearsonr, spearmanr |
| 독립표본 T검정 | `scipy.stats` | ttest_ind() |
| ANOVA | `scipy.stats` / `pingouin` | f_oneway() / pg.anova() |
| 다중회귀분석 | `statsmodels` | OLS |
| KMO / Bartlett 검정 | `factor_analyzer` | calculate_kmo(), calculate_bartlett_sphericity() |

## 4. PROCESS Macro v5 대체 대응표

| PROCESS Model | 분석 유형 | Python 구현 방식 |
|---|---|---|
| Model 4 | 단순 매개효과 | statsmodels OLS + Sobel test + Bootstrap |
| Model 7 | 조절된 매개효과 (1단계 조절) | statsmodels OLS + 상호작용항 + 조건부 간접효과 |
| Model 14 | 조절된 매개효과 (2단계 조절) | statsmodels OLS + 다단계 상호작용 + Bootstrap |
| Model 1 | 단순 조절효과 | statsmodels OLS + 상호작용항 + 단순 기울기 |
| Model 8 | 조절된 매개효과 (1,2단계 모두 조절) | statsmodels OLS + 복합 상호작용 |
| 자동 판별 | 연구모형 빌더에서 변수 관계 분석 → 해당 Model 자동 매칭 | 그래프 패턴 매칭 알고리즘 |
